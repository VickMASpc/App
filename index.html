Coleta, utilização e asseguração de dados de usuário no App Duolingo

Dados coletados

Dados fornecidos pelo usuário "Information you provide"

  • Nome (Nickname)
  • Idade aproximada
  • E-mail (Opcional)

Pesquisa de produto e desenvolvimento (Se aderido pelo usuário)

  • Informações enviadas por meio de pesquisas, entrevistas e sessões de feedback
  • Vídeo, áudio e transcrição de texto dos feedbacks

Dados de uso "Activity Data"

  • Engajamento nas atividades educacionais
  • Endereço IP (nenhuma informação de geolocalozação)

Uso dos dados

  • Fornecer, personalizar e melhorar os recursos do serviço
  • Responder a solicitações legais ou ordens judiciais
  • Enviar anúncios sobre o serviço e alterações do produto
  • Manter a segurança e proteção com dados de usuários
  • Prevenir fraudes ou outras atividades ilegais
  • Sincronizar e restaurar progresso

Riscos se houver vazamento

  • É possível que seu endereço seja rastreável por qualquer um
  • Os vídeos, áudios e transcrições de texto dos feedbacks podem conter Informações sensíveis
  • Seu E-mail estaria público, o que poderia levar á importunações e a exposição de mais dados

Como seria possível melhorar a segurança?

Melhorar a transparência com o uso dos dados para que os usuários tenham pleno conhecimento do processamento dos seus dados.

Manter as Informações e dados apenas pelo tempo necessário

Ter um tratamento específico para com os dados de usuários menor de idade, visto que, sendo um produto educacional, ofereçe serviços diretamente voltados á estes

Dicas de segurança na internet

  • Evite fornecer dados sensíveis de forma desnecessária
  • Evite redes Wi-Fi públicas, use redes seguras.
  • Use senhas fortes e não previsíveis (Se possível também altere-as regularmente)
  • Use a verificação em dois fatores (2FA) ou múltiplos fatores (MFA)
  • Verifique a integridade dos sites que acessa e fornece Informações
